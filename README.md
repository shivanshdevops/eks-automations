# launch-eks
This Automation Bash Script takes the necessary parameters and launch EKS cluster in a VPC.

It creates the below resources in a given region:
1) VPC
2) EKS Cluster Role
3) EKS Cluster
4) Key Pair for Nodes
5) Node group
6) Attaches Node group to created EKS
